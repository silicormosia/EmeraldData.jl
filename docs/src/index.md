# EmeraldData

Global datasets to feed [Emerald](https://github.com/Yujie-W/Emerald) and [silicormosia Land](https://github.com/silicormosia/Land).


## Install and use
```julia
using Pkg;
Pkg.add("EmeraldData");
```


## Tutorials
Other than the API documentation, some tutorials are provided in our online [tutorial blogs](https://silicormosia.github.io/blogs.html#category=Tutorial).
